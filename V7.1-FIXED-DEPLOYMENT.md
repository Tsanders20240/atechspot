## Cloudflare Pages environment settings

Create these under:

`Workers & Pages -> your AtechSpot project -> Settings -> Variables and Secrets`

Secret:

- `RESEND_API_KEY`

Variables:

- `FORM_TO_EMAIL=aplustechucation@gmail.com`
- `FORM_FROM_EMAIL=A+ Techucation Forms <forms@YOUR_RESEND_VERIFIED_DOMAIN>`
- `ALLOWED_HOSTS=www.atechspot.com`

Important: `FORM_FROM_EMAIL` must use a sender/domain verified inside Resend. Do not leave
`YOUR_RESEND_VERIFIED_DOMAIN` unchanged.

## Deploy

1. Extract this ZIP.
2. Upload all files to the root of the GitHub repository connected to Cloudflare Pages.
3. Replace existing files.
4. Commit: `Deploy AtechSpot V7.1 form and route fixes`
5. Wait for Cloudflare Pages deployment.
6. Purge Cloudflare cache.
7. Hard refresh with `Ctrl + Shift + R`.
8. Test:
   - https://www.atechspot.com/contact
   - https://www.atechspot.com/booking
   - https://www.atechspot.com/business
   - https://www.atechspot.com/data-recovery
9. Submit one contact form and one booking form.
10. Confirm the email arrives. If Resend is not configured, confirm the prefilled email backup opens.
