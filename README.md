## Cloudflare Pages deployment
1. Create a new Cloudflare Pages project.
2. Choose Direct Upload.
3. Upload the CONTENTS of this folder, or upload the ZIP and extract as required by your workflow.
4. Add custom domain: www.aplustechspot.com
5. Redirect the apex domain aplustechspot.com to https://www.aplustechspot.com
6. Confirm SSL is active.
7. Visit /sitemap.xml and /robots.txt.
8. Submit the sitemap to Google Search Console.
