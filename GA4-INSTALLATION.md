# GA4 Installation

Measurement ID: G-P5FFL89J6T

The Google tag is installed immediately after the opening <head> tag on every HTML page.

Cloudflare CSP now allows Google Tag Manager and Google Analytics.

Deploy:
1. Upload all extracted files to the root of the GitHub repository connected to Cloudflare Pages.
2. Replace the existing files.
3. Commit: Add GA4 tracking G-P5FFL89J6T
4. Wait for deployment.
5. Purge Cloudflare cache.
6. Open https://www.atechspot.com/?ga4=installed
7. Test in Google Analytics Realtime.

Do not add a second Google tag.
