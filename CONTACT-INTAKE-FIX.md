# Contact + Intake Fix

The Contact page now includes the complete client intake workflow.

Files included:

- `contact.html`
- `contact/index.html`

Both URLs are supported:

- `https://www.atechspot.com/contact.html`
- `https://www.atechspot.com/contact`

## Deploy

1. Replace all repository files with this package.
2. Confirm the repository contains:
   - `contact.html`
   - `contact/index.html`
3. Commit:
   `Add intake form to contact page`
4. Wait for Cloudflare Pages deployment.
5. Purge cache.
6. Test both contact URLs.
